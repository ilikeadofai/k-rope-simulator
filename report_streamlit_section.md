# 보고서 추가용: Streamlit 시뮬레이터 제작

본 탐구에서는 Python으로 작성한 K-RoPE 미니 시뮬레이터를 Streamlit 웹 앱 형태로 확장하였다. 기존 보고서에서는 일반 RoPE, 가산형 K-RoPE, 이중축 K-RoPE의 수식과 계산 결과를 정적으로 제시하였지만, Streamlit 앱에서는 사용자가 문장쌍과 회전 매개변수를 직접 조절하면서 결과 변화를 확인할 수 있도록 하였다.

앱에서 조절할 수 있는 주요 매개변수는 위치 회전 단위 $\theta$와 문법 역할 회전 단위 $\phi$이다. 일반 RoPE에서는 위치 $i$에 따라 query/key 벡터가 $i\theta$만큼 회전하고, 가산형 K-RoPE에서는 문법 역할 번호 $r_i$를 이용해 $\theta_i^K=i\theta+r_i\phi$를 적용한다. 이중축 K-RoPE에서는 위치축과 문법 역할축을 분리하여 $R^K(i)=R(i\theta)\oplus R(r_i\phi)$로 계산한다. 앱은 각 모형에 대해 attention heatmap을 표시하고, 어순이 바뀐 두 문장에서 같은 문법 관계쌍의 attention 차이 $\Delta$를 계산한다.

$$
\Delta=|a_{A,relation}-a_{B,relation}|
$$

여기서 $\Delta$가 작을수록 어순 변화 후에도 같은 문법 관계가 더 안정적으로 유지되었다고 해석한다. 또한 앱에는 RoPE의 회전 주파수 $\theta_t=10000^{-2(t-1)/d}$를 시각화하는 탭을 포함하여, 성분쌍마다 서로 다른 회전 속도가 위치 정보를 어떻게 다층적으로 표현하는지 확인할 수 있도록 하였다.

이 시뮬레이터는 실제 LLM을 학습시키거나 성능을 증명하는 도구가 아니라, RoPE와 K-RoPE의 수학적 구조를 고등학교 대수 수준에서 이해하기 위한 교육용 도구이다. 따라서 사용자 정의 문장 기능에서도 자동 형태소 분석을 사용하지 않고, 어절과 문법 역할을 직접 입력하도록 설계하였다. 이는 앱이 한국어 문장 분석기처럼 과장되어 해석되는 것을 피하고, 본 탐구의 핵심인 삼각함수 회전과 attention 계산 구조에 집중하기 위한 선택이다.
